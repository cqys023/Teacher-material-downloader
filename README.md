# Teacher-material-downloader
# [国家中小学智慧教育平台 电子课本](https://basic.smartedu.cn/tchMaterial/)下载工具

> [!TIP]
> 因国家中小学智慧教育平台的电子教材只能在线观看，不能下载，这样给广大读者带来不便，为方便需要教材电子版本的读者，本作者特开发了这个小程序。
## ✨工具特点

- 📚**界面干净简洁**：下载工具极简，无任何广告插件。
- 📚**支持预览功能**：能预览即将下载 PDF 课本文件是否为最新或是自己想要的课本。
- 📂**自动文件命名**：程序会自动使用教材名称作为文件名，方便管理下载的课本文件。
- 🖥️**高 DPI 适配**：优化 UI 以适配高分辨率屏幕，避免界面模糊问题。
- 📊**下载进度可视化**：实时显示下载进度，支持暂停/恢复操作。
- 💻**平台支持**：目前仅支持 Windows（图形界面），后期如使用人数增多会考虑其他平台版本发布。

## 📥下载与安装方法
本项目的 [GitHub Releases 页面](https://github.com/happycola233/tchMaterial-parser/releases)直接进行下载。
在下载完成之后，解压即可运行本程序，不需要额外的安装步骤。

## 🛠️使用方法
### 1.打开国家中小学智慧教育平台：[https://basic\.smartedu\.cn](https://basic.smartedu.cn)，然后选择：教材。如下图：

![](images/image_f073e39f485f4fefb0d17e41c81fa332.png)

进入后点击右上角登录，如未注册的就注册个账号，免费注册的，因为后面预览教材必须要登录才能预览。

![](images/image_a485a96399f448fbaeaf774225518e66.png)

### 2.注册后登录，然后点击要下载的教材，我这随便选择了一本，如下图：

![](images/image_ac5a0eba235c4bf0a61d71e41356060f.png)

### 3.点击教材后进入预览界面：

![](images/image_f3d221c64ce94d008a3b9baad8df603c.png)

### 4.在这个页面中需要进入开发者模式，按F12或在浏览器中选择更多工具，开发人员工具。如下图：

![](images/image_1fff7a6c3f1642778418bf1b56976dbf.png)

进入开发者模式后，点击全部，然后点击下浏览器的刷新按钮，找到viewer\.html?file=后面一长串地址。如下图这个地址：

![](images/image_7ae0374ed1b842e4bc7f42044155a31d.png)

### 5.复制上面这个地址，粘贴到软件教材地址栏中，然后点击下载即可，下载后的pdf电子文件默认保存在软件目录下。如下图：

![](images/image_7bafb3c9498348f6a548d4e686dc4c60.jpeg)

![](images/image_77b997a58a5845ea98c8301492f7dca9.png)

### 6.下载完成后的文件在这里：

![](images/image_79276df9079c468f8196bd1b0658b471.png)
